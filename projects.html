Cleanliness comes first when it comes to using Washroom. This Project takes care of that. An app linked to the smart washroom which has special hardware features to detect odour of the washroom, auto flush, sensor based basin, automated commode etc. In case addition public are not satisfied with the facilities available in the washroom, they can send feedback to the respective authorities.
This project is divided into 2 different apps, one for the public and one for the authorities.
